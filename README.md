# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) uses [Oxc](https://oxc.rs)
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) uses [SWC](https://swc.rs/)

## React Compiler

The React Compiler is not enabled on this template because of its impact on dev & build performances. To add it, see [this documentation](https://react.dev/learn/react-compiler/installation).

## Expanding the ESLint configuration

If you are developing a production application, we recommend using TypeScript with type-aware lint rules enabled. Check out the [TS template](https://github.com/vitejs/vite/tree/main/packages/create-vite/template-react-ts) for information on how to integrate TypeScript and [`typescript-eslint`](https://typescript-eslint.io) in your project.
# AI Career Hub 🚀

AI Career Hub is an AI-powered career guidance platform that helps students plan their careers, improve resumes, prepare for interviews, and follow personalized learning roadmaps.

## Features

### 📄 Resume Analyzer

* Upload resume
* AI-powered ATS score
* Resume improvement suggestions

### 🎤 AI Interview Prep

* Generate interview questions
* Get AI feedback on answers
* Practice for job interviews

### 🗺️ Roadmap Generator

* Personalized learning roadmaps
* Multiple career paths
* Step-by-step learning guidance

### 📊 Progress Tracking

* Track learning progress
* Monitor completed tasks
* Learning streak system

### 🕒 Saved History

* View previous sessions
* Resume analysis history
* Interview practice history

## Tech Stack

### Frontend

* React.js
* Tailwind CSS
* React Router
* Axios

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose

### AI Integration

* Google Gemini AI

## Installation

### Clone Repository

```bash
git clone https://github.com/au422523205006/ai-career-hub.git
cd ai-career-hub
```

### Install Dependencies

Frontend:

```bash
npm install
```

Backend:

```bash
cd backend
npm install
```

### Run Project

Frontend:

```bash
npm run dev
```

Backend:

```bash
npm run server
```
